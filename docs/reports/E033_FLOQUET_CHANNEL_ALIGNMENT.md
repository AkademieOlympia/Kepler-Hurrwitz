# E-033 Floquet Channel Alignment Report

**Register-ID:** `E-033` (Energiedoku / Kepler-Zeit-Block)  
**Lean-Kollisionshinweis:** Lean-internes Label **E-033** in `PrimvierlingSymmetry.lean` ist **E-046** im globalen Register. Dieser Report bezieht sich ausschließlich auf die Python-Floquet-Annotation in `kepler_eabc_atlas.py`.

## Status

**[B/C] alignment only.**

This alignment report measures whether the observed ΔM tails are numerically compatible with the formal 2×4 channel lift. It does not prove that the channel lift generates the ΔM spectrum.

**Not claimed:** Delta-M values are not proven to realize or be generated by the channel projection.

Lean side:

- period relation: `8 = 2 × 4` (`lift_sheet` 0/1 over chi phases 0–3)

## Numerical Annotation

Python side (mirror of the Lean atlas, descriptive only):

- `floquet_step_channel(step)` — channel label for Floquet step index
- `annotate_delta_m_with_channels(...)` — attach channel / chi_phase / lift_sheet to a ΔM tail
- `channel_alignment_summary(...)` — per-channel count, mean, population std (`statistics.pstdev`), min/max, sign pattern
- `lift_sheet_pair_differences(...)` — compare steps `i` vs `i+4`; missing pairs are skipped
- `max_lift_sheet_abs_difference(...)` — max `|ΔM_{i+4} − ΔM_i|` over complete pairs (None if none)
- `summarize_annotated_delta_m(...)` — per-scenario governance bundle for JSON export

## Main Diagnostics

Input: `kepler_time_bridge` scenarios (500 steps, tail=64). Each scenario record carries its own `alignment_summary` with `status`, `not_claimed`, channel stats, sheet-pair differences, and `max_lift_sheet_abs_difference`.

Per scenario, inspect in `docs/energiedoku_exports/kepler_eabc_floquet_annotation.json`:

- `alignment_summary.lift_sheet_pair_differences` — `|ΔM_i − ΔM_{i+4}|` per chi phase
- `alignment_summary.max_lift_sheet_abs_difference` — headline compatibility metric
- `alignment_summary.status` / `not_claimed` — per-record governance (not only global)

Human-readable tables: `docs/energiedoku_exports/e033_floquet_channel_alignment.md`.

## Interpretation

Small sheet-pair differences support compatibility with the formal 2×4 lift indexing. Large differences indicate that the lift is currently only an indexing atlas — the channel labels annotate steps but do not generate the observed ΔM spectrum.

Baseline cyclic shows sheet pairs with opposite sign (±1.85619), hence max ≈ 2× line spacing — consistent with a two-line spectrum, not sheet degeneracy. Only `perturbation_at_100` shows zero sheet-pair spread; that reflects the perturbed tail structure, not a proof that the channel projection realizes ΔM.

Evidence status remains **[B/C]** until multi-scenario stability of small sheet differences or a consistent channel/sign profile is established under explicit governance.
