# L4: Holography as Structural Inspiration

Dieses Dokument ordnet holographische Begriffe als reines
Strukturmotiv ein. Es fuehrt **keine** neue Evidenzstufe ein.

## Defensiver Rahmen

Nicht behauptet wird:

- `EABC == AdS/CFT`
- Ersatz etablierter Quantengravitationstheorien

Stattdessen:

- Holographie als Strukturprinzip der Randkodierung
- Frage nach `Boundary Encoding` im EABC-Sprachraum

## Kernidee

`Volumen -> Oberflaeche -> Randinvariante -> rekonstruierbare Innenstruktur`

Die methodische Frage lautet:
Welche Randdaten reichen aus, um innere Freiheitsgrade zu rekonstruieren
oder wenigstens hart zu beschraenken?

## Struktur-Mapping

| Holographie / AdS-CFT | EABC-Struktur |
|---|---|
| Bulk | innerer Zustandsraum / Normschalen |
| Boundary | Rand-/Oberflaechenkanaele |
| CFT-Daten | diskrete Observablen / Signaturen |
| Radiale Richtung | Skalen-/Schalenrichtung |
| RG-Fluss | Renormierungskette / Retraktion |
| Entanglement-Wedge | rekonstruierbarer Teilraum aus Randdaten |
| Holographisches Prinzip | Informationsreduktion auf Randinvarianten |

## Oberflaechenhierarchie (motivisch)

`S^1 subset S^2 subset S^3 subset ... subset S^7`, mit
`boundary(B^(n+1)) = S^n`.

| Objekt | Rolle |
|---|---|
| `S^1` | Phase / Umlauf / Holonomie |
| `S^2` | Richtungsraum / Ikosaeder-Motive |
| `S^3` | Quaternionen / `SU(2)` |
| `S^7` | Oktonionische Struktur |
| `B^8` bzw. `R^8` | 8D-Signaturraum / hochdimensionaler Zustandsraum |

## Leitfrage fuer das EABC-Programm

Kann ein hochdimensionaler Zustandsraum (etwa ein 8-dimensionaler
Signaturraum) so organisiert werden, dass Randdaten/Oberflaechenstrukturen
wesentliche Informationen ueber innere Freiheitsgrade kodieren?

## Quellenhinweis auf L4-Niveau

- Susskind: Holographisches Prinzip als Flaechen-/Randkodierung
- Maldacena: AdS/CFT als Beispiel einer Bulk/Boundary-Dualitaet

Im EABC-Kontext dienen diese Referenzen ausschliesslich der
mathematischen Orientierung.
