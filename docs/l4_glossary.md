# L4 Glossary

Dieses Glossar vereinheitlicht die Projektbedeutung zentraler Begriffe auf
L4-Ebene. Es ist terminologische Hilfe, keine Evidenzquelle.

| Begriff | Projektbedeutung |
|---|---|
| Verbindung | Transportregel zwischen Zustaenden |
| Holonomie | Umlaufinvariante entlang geschlossener Zyklen |
| Defekt | lokale Verletzung einer Referenzstruktur |
| Retraktion | Rueckfuehrung auf kanonischen Zustand/Teilraum |
| Attraktor | stabiler Zielbereich der Dynamik |
| Projektion | Wahl eines Repräsentanten aus einer Klasse |
| Observable | auswertbare Invariante oder Messgroesse |
| Symmetrie | Gruppenwirkung auf Zustaenden/Operatoren |
| Dualitaet | aequivalente Beschreibung derselben Struktur |
| Holographie | Randdaten kodieren Innenstruktur |
| Konfinierung | Einschraenkung auf stabilen Bereich/Zustandssektor |
