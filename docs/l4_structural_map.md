# L4 Structural Map

Dieses Dokument ordnet die L4-Inspirationen in eine gemeinsame Taxonomie.
Es beschreibt Strukturprinzipien, nicht Evidenzgrade.

Uebergreifender Rahmen: `l4_unified_structural_framework.md`  
Terminologie: `l4_glossary.md`

## Strukturprinzipien

| Strukturprinzip | Mathematik | Beispiele im Projekt |
|---|---|---|
| Symmetrie | Gruppen, Invarianten | EABC-Orbits, `PrimvierlingSymmetry`, `CyclicWordOrbit` |
| Geometrie | Mannigfaltigkeiten, Sphaeren, Signaturraeume | `S^1`, `S^2`, `S^3`, `S^7`, `R^8`-Signaturraum |
| Transport | Verbindungen/Transportregeln | Uebergangsoperatoren, Green-Kernel, Bridge-Operatorik |
| Kruemmung/Defekt | Defektoperatoren, Umlaufstrukturen | Defektoperator, Umlaufdefekt, Retraktionsdefekt |
| Spektrum | Eigenwerte/Spektralfunktionen | Dodekaeder-/Hurwitz-Spektren, Slice-Loci |
| Thermodynamik | Zustandssummen, Entropie, freie Energie | `l4_thermodynamic_structural_inspiration.md` |
| Holographie | Rand/Bulk-Kodierung | `l4_holography_structural_inspiration.md` |
| Diskrete Dynamik | Iteration, Retraktion, Attraktorbildung | Collatz-/Odd-Core-Dynamik, B11-Attraktor |

## Theoriequelle -> Strukturprinzip -> EABC-Nutzung

| Theoriequelle | Strukturprinzip | EABC-Nutzung |
|---|---|---|
| Aharonov-Bohm | Verbindung / Holonomie | Transport, Defektumlauf |
| Holographie | Bulk/Boundary | Randdaten beschraenken Innenstruktur |
| Thermodynamik | Spektrum / Zustandssumme | Entropie, freie Energie als Strukturparameter |
| Stringtheorie | Dualitaet / Kompaktifizierung | Projektion hoeherdimensionaler Signaturen |
| Loop Quantum Gravity | diskrete Geometrie | Graphen, Labels, Flaechen-/Volumenstruktur |
| Quantenteleportation / Bell | Verschraenkung, Korrelation, Rekonstruktion | nicht-faktorisierbare Signaturzustaende, Projektion, Transferkanaele |

## Querverbindung Thermodynamik <-> Holographie (L4-Schema)

`Spektrum -> Zustandssumme -> Entropie -> Randinformation`

Das ist ein Organisationsschema fuer spaetere Fragestellungen und keine
physikalische Identifikation.

## Schutzsatz

Die genannten Theoriequellen werden ausschliesslich als mathematische
Strukturinspiration genutzt. Aus dieser Zuordnung folgt keine physikalische
Identifikation des EABC-Modells mit einer etablierten Theorie.

## Methodischer Hinweis

- L4 mappt Strukturprinzipien.
- Evidenz bleibt ausschliesslich in `[A]/[B]/[C]/L4` mit Registerregeln.
- Kein Strukturprinzip erzeugt fuer sich allein eine Evidenzaufwertung.
