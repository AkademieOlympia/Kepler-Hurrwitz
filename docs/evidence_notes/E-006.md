## Beweisidee (endlich, mod 12)

Die lokale Aussage wird als endliche Restklassenrechnung gefasst:

1. `r = n % 12` impliziert `r ≤ 11`.
2. Aus `n % 2 = 1` folgt die ungerade Mod-12-Fallunterscheidung
   `r ∈ {1,3,5,7,9,11}`.
3. Damit gilt direkt `B11Channel r` (`r ≤ 11 ∧ r % 2 = 1`).

Damit ist der Upgrade-Fall `E-006: [C] -> [A]` formal geschlossen.
