# Project Glossary

Dieses Glossar vereinheitlicht die wichtigsten im Projekt verwendeten Begriffe.
Es dient der Lesbarkeit, nicht der Evidenzklassifikation.

## Governance und Evidenz

| Begriff | Bedeutung |
|---|---|
| Evidenzklasse `[A]` | formal abgesichert (Lean) |
| `A-T` | formal bewiesenes Theorem |
| `A-D` | formale Definition (ohne eigenen Theoremschritt) |
| `A-I` | formale Infrastruktur / Interface |
| Evidenzklasse `[B]` | reproduzierbare numerische Evidenz |
| Evidenzklasse `[C]` | offene mathematische Hypothese/Interface |
| Ebene `L4` | programmatische, strukturelle Interpretation |
| `Depends on` | logische/technische Abhaengigkeit |
| `Supports` | wissenschaftliche Unterstuetzung ohne Beweisimplikation |
| Stabilitaet `stable` | langfristig tragender Baustein |
| Stabilitaet `experimental` | kann sich mit neuen Resultaten aendern |
| Stabilitaet `deprecated` | nicht mehr aktiv genutzt, aber dokumentiert |
| Stabilitaet `superseded` | durch neueren Eintrag ersetzt |

## Kernmathematik (EABC/Hurwitz/Collatz)

| Begriff | Bedeutung |
|---|---|
| EABC-Signatur | diskrete Mehrkanal-Signatur eines Zustands |
| Hurwitz-Struktur | arithmetische/quaternionische Operator- und Primstruktur |
| Metakommutation | nichttriviale Vertauschung im Hurwitz-Kontext |
| Odd Core | ungerader Kern nach Entfernen des 2-adischen Anteils |
| Retraktion | Rueckfuehrung auf kanonischen Zustand/Teilraum |
| Attraktor | stabiler Zielbereich der Dynamik |
| Schalenkanal | diskrete Einordnung nach `nu2(3m+1)`-Tiefe |
| Bremensaal | EABC-Dreierblock auf dem 12-Ecken-Ikosaeder-Träger (Projekt „Die drei Musketiere“) |
| Kanonisches Embedding | Gauge-fixierte EABC→Ikosaeder-Abbildung via K1–K4 (lex-sort, mod-4-Orbit, E-Mittelachse, chi-Label) |
| Musketiere-Nachbar-Dreier | Graph-Dreieck mit Labels der drei Nicht-Träger-Kanaele, das den Träger-Saal beruehrt |
| Defektoperator | Operator zur Erfassung lokaler Nichttrivialitaet |
| Umlaufdefekt | Defekt, der sich erst entlang geschlossener Umlaeufe zeigt |
| Orbit | unter einer Gruppenwirkung erzeugte Zustandsklasse |

## Spektral-/Interferenzbegriffe

| Begriff | Bedeutung |
|---|---|
| Spektrum | Menge/Struktur relevanter Eigenwerte |
| Slice-Locus | algebraische Ortsmenge im reduzierten Parameterraum |
| Interferenzpunkt | kanonische Schnittstelle mehrerer Locus-Bedingungen |
| Nichtfaktorisierbarkeit | Zustand nicht als Produkt lokaler Komponenten zerlegbar |
| Holonomie | Umlaufinvariante aus Transport-/Verbindungsstruktur |
| Invariante | unter zugelassener Transformation unveraendert |

## L4-Strukturprinzipien

| Begriff | Bedeutung im Projekt |
|---|---|
| Verbindung | Transportregel zwischen Zustaenden |
| Kruemmung/Defekt | Nichttrivialitaet der Verbindungsstruktur |
| Bulk/Boundary | Innenraum-/Randkodierungs-Perspektive |
| Dualitaet | aequivalente Beschreibungen derselben Struktur |
| Zustandssumme | gewichtete Summe ueber Spektralzustaende |
| Entropie | Mass effektiver Zustandsvielfalt |
| freie Energie | Konkurrenzgroesse von Energie- und Entropiebeitrag |
| Korrelation | nichtlokale bzw. gekoppelte Strukturinformation |
